HUNTER 16 - GET DISKETTE PORT

AX = 8400h.

BH = Drive (0:
A:, 1: B:)


Return:
AL = Port (0: COM1, >0: COM2)

Desc: Return the COM port used for the floppy drive

See Also: AX=8401h

Category: Vendor-specific BIOS Extensions - Int 15h - H
