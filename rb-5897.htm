CauseWay - AliasSel - CREATE READ/WRITE DATA ALIAS SELECTOR

AX = FF06h
BX = original selector


Return:
CF clear if successful
AX = new data selector aliasing original selector
CF set on error

See Also: AX=FF03h - AX=FF04h
