Banyan VINES - SECONDARY 3270 INTERFACE

AX = 000Ah

Note: Either 3270/SNA or 3270/BSC interface will use AX=000Ah, depending on which is loaded second. The first interface loaded will use AX=0002h

See Also: INT 61/AX=0002h
