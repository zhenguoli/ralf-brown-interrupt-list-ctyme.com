Novell NetWare - BINDERY SERVICES - CREATE PROPERTY

AH = E3h subfn 39h
DS:SI -> request buffer (see #01934)
ES:DI -> reply buffer (see #01935)


Return:
AL = status (see #01933)

Note: This function is supported by Advanced NetWare 1.0+ and Alloy NTNX

See Also: AH=E3h/SF=32h - AH=E3h/SF=3Bh - AX=F217h/SF=39h

EFh    invalid name

F6h    not permitted to delete properties
F7h    not permitted to create properties
FBh    no such property
FCh    no such object
FEh    server bindery locked
FFh    bindery failure
