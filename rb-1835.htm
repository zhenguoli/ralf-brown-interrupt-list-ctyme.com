HP Vectra EX-BIOS - F16_KBD - GET KEYBOARD INFORMATION

AX = 6F08h


Return:
AH = status
00h successful
02h unsupported (non-HIL, i.e. standard, keyboard)
BH = HP-HIL address (HP Vectra AT only???)
BL = HP-HIL ID (HP Vectra AT only???)
BL = keyboard language (ES/QS/RS only???) (see #00603)

Notes: Supported by the original HP Vectra AT and ES/QS/RS series HP Vectras. The driver's address in the HP_VECTOR_TABLE (see INT 6F/AH=00h) may be computed as (BH-1)*6 + N, where N is the address of the first HP-HIL device driver (see INT 6F/AH=0Ah"F_INQUIRE_FIRST")

See Also: AX=6F05h - AX=6F09h - INT 6F/AH=0Ah"F_INQUIRE_FIRST"
