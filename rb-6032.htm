TRUEDOX Mouse driver v4.01 - GET/SET HARDWARE PARAMETERS

AX = 006Ch
BX = new IRQ (0003h or 0004h), or 0000h to get current values only
CL = new IRQmask (sent to 8259)
DX = new base I/O port


Return:
BX = current IRQ
DX = light pen state???

Note: This is the mouse driver for the Dell Dimension series of computers, by TRUEDOX Technology Corporation

See Also: AX=00A1h - AX=0666h

Category: Mouse/Pointing Device - Int 33h - T
