DESQview - APPNUM - GET CURRENT PROGRAM'S NUMBER

AX = DE07h


Return:
AX = number of program as it appears on the "Switch Windows" menu

Note: This API call may be made from a hardware interrupt handler

See Also: AX=DE00h
