MS Windows - WINDOWS ENHANCED MODE & 286 DOSX EXIT BROADCAST

AX = 1606h
DX = flags
bit 0 = 0 if Windows enhanced-mode exit
bit 0 = 1 if Microsoft 286 DOS extender exit
bits 1-15 reserved (undefined)

Notes: If the init broadcast fails (AX=1605h returned CX <> 0), then this broadcast will be issued immediately.. This call will be issued in real mode. Novell DOS v7.0 (Update 8 - Update 15) TASKMGR in multitasking mode uses this broadcast, even if TASKMGR.INI sets WinPresent= to OFF

See Also: AX=1605h - AX=1609h
