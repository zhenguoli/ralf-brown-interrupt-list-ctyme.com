DESQview v2.00+ - FINDMAIL - FIND MAILBOX BY NAME

AX = DE0Eh
ES:DI -> name to find (see #00551)
CX = length of name


Return:
BX = 0000h not found
0001h found
DS:SI = object handle

See Also: AH=12h/BH=11h - AH=12h/BH=12h"GETNAME"
