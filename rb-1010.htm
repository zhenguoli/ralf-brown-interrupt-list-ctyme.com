COMM-DRV v14.0 - READ PORT METRICS - GET ERROR CODE AND BUFFER STATUS

AX = 8000h
DX = port number


Return:
AX = code for last error (see #00357)
BX = number of characters in output buffer
CX = nubmer of characters in input buffer
DX = state flag (see #00358)

Program: COMM-DRV is a universal serial communications driver by Willies' Computer Software Company, which supports standard INT 14 and FOSSIL calls as well as its own interfaces

See Also: AX=8001h"COMM-DRV" - AX=8002h"COMM-DRV" - AX=8003h"COMM-DRV"
