Chips & Technologies '65530' BIOS - MONITOR SENSING HOOK

AX = 5F35h


Return:
DL = boot display
00h CRT
01h flat panel (LCD)
02h both simultaneously
leave unchanged to boot according to BIOS settings

See Also: AX=5F31h - AX=5F33h - INT 10/AX=5F50h
