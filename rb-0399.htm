HUNTER 16 - SET LCD WINDOWS POSITION

AH = 74h
DH = row (0..24)
DL = column (0..79)


Return:
Nothing

Desc: Set the position in the virtual screen being displayed in the LCD screen. If Split Screen (INT 10/AH=66h) is used, this call sets the position for the window which has the cursor.

See Also: AH=66h"HUNTER" - AH=75h - AH=76h

Category: Video - Int 10h - H
