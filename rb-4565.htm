MS Windows - VCOND - GET API ENTRY POINT

AX = 1684h
BX = 0038h (virtual device ID for VCOND device) (see #02642)
ES:DI = 0000h:0000h


Return:
ES:DI -> VxD API entry point (see #02678)
0000h:0000h if the VxD does not support API in current mode
