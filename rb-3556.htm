Novell NetWare - FILE SERVER - ENABLE TRANSACTION TRACKING

AH = E3h subfn D0h
DS:SI -> request buffer (see #02021)
ES:DI -> reply buffer (see #02020)


Return:
AL = status (00h,C6h) (see #02025)

Desc: Restart transaction tracking after being stopped either explicitly by AH=E3h/SF=CFh or automatically due to a full transaction volume

Notes: This function is supported by Advanced NetWare 2.1+. The calling workstation must have console operator privileges

See Also: AH=E3h/SF=C8h - AH=E3h/SF=CFh - AX=F217h/SF=D0h 


Format of NetWare "Enable Transaction Tracking" request buffer:

Offset  Size    Description     (Table 02021)
00h    WORD    0001h (length of following data)
02h    BYTE    D0h (subfunction "Enable Transaction Tracking")

See Also: #02020 - #02019

Category: Netware - Int 21h - N
