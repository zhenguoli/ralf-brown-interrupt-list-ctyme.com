Emulaser ELTSR - PRINT CAPTURE FILE???

AH = 09h
BX = ???
CX = ???
DX = printer port (01h COM1, 02h COM2, 05h LPT1, 06h LPT2)


Return:
AX = status
00h successful
FFh failed

Program: ELTSR is the resident portion of the Emulaser PostScript emulator by Vertisoft Systems, Inc.

Note: This function calls through to INT 1A/AX=E401h, and thus requires that either ELSPL or Disk Spool II be installed

See Also: AH=0Ah - INT 1A/AX=E401h - INT 1A/AH=E5h
