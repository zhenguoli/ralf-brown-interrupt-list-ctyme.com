Alloy NTNX,MW386 - RESET SPOOLER

AH = 8Dh
AL = 00h


Return:
AL = status (see #00642)

Notes: Clears all buffers and resets spooler to boot-up values. MW386 supports this function for compatibility only; it is a NOP
