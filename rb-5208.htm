WinDOS v2.11 - GET OTHER MAIN SHELL PARAMETERS

AX = AF0Ch


Return:
DI:AX = stack
SI = offset of break handler

Desc: Retrieve the parameters last set by AX=AF06h (normally by the main command interpreter)

See Also: AX=AF00h"WinDOS" - AX=AF06h"WinDOS"
