Banyan VINES - GET ADDRESS OF GLOBAL VARIABLE

AX = 0006h.

BX = bits 14..0:
Request ID (see #03431)

bit  15:
Result destination flag (0 for ES:BX, 1 for AX:BX)


Return:
AX = status
0000h successful
000Bh invalid request
ES:BX -> internal variable (bit 15 of BX was not set)
AX:BX -> internal variable (bit 15 of BX was set)

Note: This service is handled by the redirector

See Also: AX=0004h/SF=0013h - AX=0004h/SF=0014h
