Screen Thief v1.00 - RELOCATED IRQ0

Range:
INT 78h to INT E0h, selected by commandline switch

Note: Screen Thief relocates IRQs 0 through 7 to INT D8 to INT DF by default, but may be directed via a commandline switch to use any range starting at a multiple of 8 between 78h and E0h
