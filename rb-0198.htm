Cirrus Logic BIOS - GET USER OPTIONS

AH = 12h
BL = 9Ah


Return:
AX = options word 1 (see #00031)
CX = options word 2 (see #00032)
BX,DX reserved

See Also: AH=12h/BL=80h - AH=12h/BL=84h - AH=12h/BL=93h - See Also: AH=12h/BL=A3h - AH=12h/BL=A4h - #00732 at INT 1A/AX=B102h
