PCMCIA Socket Services v1.00 - GET NUMBER OF ADAPTERS

AH = 80h


Return:
CF clear if successful
CX = 5353h ('SS') if Socket Services installed
AL = number of adapters present (0-16)
AH destroyed
CF set on error
AH = error code (see #00656)

See Also: AH=83h"PCMCIA"
