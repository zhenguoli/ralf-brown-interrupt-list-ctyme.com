LIM EMS - GET MANAGER STATUS

AH = 40h


Return:
AH = status (00h,80h,81h,84h) (see #03648)

Note: This call can be used only after establishing that the EMS driver is in fact present

See Also: AH=3Fh - AX=FFA5h - @xxxxh:xxxxh"PMM"
