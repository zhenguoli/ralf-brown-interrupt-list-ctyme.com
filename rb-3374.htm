Novell NetWare - SYNCHRONIZATION SERVICES - CLEAR FILE SET

AH = CFh


Return:
AL = 00h

Desc: Unlock and remove all files from log table

Note: This function is supported by NetWare 4.0+, Advanced NetWare 1.0+, and Alloy NTNX
