NOW! v3.05 - GET INFORMATION

AX = 7B00h
CX:DX -> 1F8h-byte buffer for information record (see #00288)


Return:
AX = 0000h
BX = segment of main resident code

Program: NOW! is a disk cache by Vertisoft Systems, Inc.

See Also: AX=7B02h - AH=EFh
