Communicating Applications Specification - SUBMIT A TASK

AX = CB01h
DS:DX -> ASCIZ name of task control file


Return:
AX >= 0: event handle.

AX < 0:
Two's complement of error code (see #03048)

Note: Files needed for an event must be kept until task is complete or error

See Also: AX=CB0Bh - AX=CB15h
