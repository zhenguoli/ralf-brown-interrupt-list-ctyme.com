DECnet DOS LOCAL AREA TRANSPORT - OPEN SESSION

AH = D0h
DH = FFh
AL = password flag
FFh no password
0Fh password at ES:DI
ES:BX -> LAT session control block (see #03727)
ES:DI -> 16-byte blank-padded password (optional)


Return:
AH = 00h success
DL = handle

See Also: AX=D000h
